Интеграции

Свяжите WhatsApp Cloud API с вашим аккаунтом, чтобы входящие сообщения попадали во Входящие.

WhatsApp

Скопируйте Phone number ID из кабинета Meta (раздел WhatsApp → API Setup / Номер) и вставьте ниже. Он должен совпадать с metadata.phone_number_id во входящих вебхуках.

Telegram AI‑бот (отдельный бот для каждого бизнеса)

Арендатор создаёт бота в BotFather и вставляет токен. После сохранения получите уникальный URL вебхука (только для этого аккаунта) и подключите его через setWebhook.

Хранится в вашей БД. Не публикуйте токен и не коммитьте его в git.
Примеры: Europe/Moscow, Europe/Prague.
